
# Architecture

## Default Project Pattern

To create a default project pattern, you can follow the next folder structure:

```text
app
├── login
│   ├── page.ts
└── dashboard
    └── questions
        └── create
            └── page.tsx
        └── page.tsx
    ├── page.tsx
    └── layout.tsx
└── lib
    ├── actions.ts (server action functions)
    ├── definitions.ts (types)
    └── data.ts (db functions)
└── ui
    ├── global.css
    └── fonts.ts
└── error.tsx    
└── page.tsx    
└── layout.tsx    
```
